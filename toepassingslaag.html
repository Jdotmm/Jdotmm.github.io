Toepassingsprogramma's

Toepassingsprogramma’s zijn applicaties die zijn gemaakt om een specifieke functie uit te voeren, toepassingsprogramma’s hebben een specifieke functie die de behoeften van gebruikers vervult. Je hebt dus niet een soort toepassingsprogramma, je hebt bijvoorbeeld Microsoft Word, Google Chrome, Gmail of Spotify. Games en applicaties zoals Discord of Skype zijn ook voorbeelden van toepassingsprogramma’s. Iedereen gebruikt dus wel toepassingsprogramma’s, want dat zijn de programma’s die we moeten gebruiken om de computer te laten doen wat we willen, zonder zou je deze tekst hier niet eens kunnen lezen.

Databases

Een database is een verzameling van gegevens die op een geordende manier gerangschikt zijn. Hoewel je het misschien niet door hebt maakt eigenlijk iedereen wel gebruik van databases, social media maakt namelijk gebruik van databases om foto's, video's en andere gegevens op te slaan. Als je een mailtje stuurt of leest maak je ook gebruik van databases, al e-mails en contactgegevens worden opgeslagen in een database.

API's

Een API (Application Programming Interface) is een set regels en protocollen waarmee verschillende softwaretoepassingen met elkaar kunnen communiceren en informatie kunnen uitwisselen. API’s maken communicatie tussen verschillende softwarecomponenten mogelijk, bijvoorbeeld dus tussen databases, applicaties en het besturingssysteem. Door middel van een API kunnen programma’s dus gegevens met elkaar delen en wisselen. Het gebruik van een API zorgt er ook voor dat applicaties die in een andere taal geschreven zijn met elkaar kunnen communiceren, hierdoor kan je dus makkelijk applicaties van andere mensen in je eigen programma implementeren.
