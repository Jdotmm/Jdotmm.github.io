Dit plaatje laat wat verschillen zien tussen bitmap en vector afbeeldingen


Het belangrijkste verschil tussen de twee is dat een bitmap afbeelding uit pixels (kleine vierkantjes met 1 bepaalde kleur) en een vectorafbeelding bestaat niet uit dit soort vietkantjes, maar vooral uit wiskundige formules, het gevolg hiervan is dat als je inzoomt op een bitmap afbeelding dat deze onscherp wordt, een vectorafbeelding heeft dit probleem niet

Bitmap afbeeldingen

Een bitmap afbeelding heeft 2 belangrijke eigenschappen, dit zijn echter ook meteen twee nadelen aan het gebruik van bitmapafbeeldingen. Het eerste nadeel is dat een bitmapafbeelding veel opslagruimte kost, het tweede nadeel is dat als je inzoomt op een bitmapafbeelding de losse pixels zichtbaar worden, als je inzoomt is je afbeelding dus niet meer scherp.

Vector afbeeldingen

Een vectorafbeelding is niet opgebouwd uit pixels maar uit de vormen waaruit de afbeelding is opgebouwd. Een vectorafbeelding is dus erg handig als je een afbeelding heel groot ergens gaan laten zien, zoals op een reclamebord. Als je een bitmapafbeelding heel groot nogsteeds scherp wilt hebben kost dat heel veel pixels en dit zou dus heel veel opslag ruimte in nemen, daarom worden vectorafbeeldingen dus vaker gebruikt voor hele grote afbeeldingen. Wat wel een nadeel is aan een vectorafbeelding is dat je alleen vormen op kan slaan een foto bestaat niet uit deze vormen, je kan normale foto's dus niet als een vectorafbeelding opslaan.
