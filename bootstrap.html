Over mijn site

Dit is mijn site die ik heb gemaakt voor informatica, op deze site kan je informatie vinden over bepaalde onderwerpen die horen bij informatica. De site is geprogrammeerd met HTML en CSS. Om dingen duidelijker te maken heb ik ook gebruik gemaakt van afbeeldingen en embeds en videos.

Bootstrap 5

Bootstrap is een verzameling hulpmiddelen voor het maken van websites en webtoepassingen. Het is gratis en open source. Het bevat sjablonen gebaseerd op HTML en CSS voor typografie, formulieren, knoppen, navigatie en andere interfaceonderdelen. Het bevat ook JavaScript-extensies.
